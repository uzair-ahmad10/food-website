# FoodHub Website

A responsive, static landing page for a food delivery service named **FoodHub**. Built with standard HTML and CSS.

## Features
- **Hero Section**: An engaging introductory section with a clear call to action ("Order Now").
- **Category Filter**: A sidebar to select food categories like Turkish, Pakistani, Chinese, Indian, Italian, and Japanese cuisines.
- **Menu Items**: A grid displaying various meals with details including images, names, ratings, and pricing (old vs. new price comparisons).
- **Order Form**: A simple, user-friendly form section for customers to enter details and place their orders.
- **Footer**: Includes site copyright information and standard social media links.

## Technologies Used
- **HTML5**: For semantic page structure.
- **CSS3**: For layout, styling, and visual aesthetics.
- **FontAwesome**: For social media icons.

## File Structure
- `index.html`: Main HTML page containing the layout and content.
- `style.css`: Stylesheet handling colors, typography, sizing, and arrangement of elements.
- `images/`: Directory storing all the graphical assets and food item pictures.

## How to View
To view this project locally:
1. Clone or download the repository to your local machine.
2. Navigate to the project directory
3. Open `index.html` in any modern web browser (Google Chrome, Mozilla Firefox, Safari, Edge, etc.). No local development server or additional setup is required.
4. Or simply [click here]url(https://foodhub-website.netlify.app/)
